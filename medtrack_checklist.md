# ЧЕК-ЛИСТ ВЕРИФИКАЦИИ ДЛЯ ТРЕБОВАНИЯ #2

## "Система должна предупреждать об опасных значениях"

### Требование после уточнения:
"При показателях вне нормы (давление > 180/110, пульс > 120) система отправляет push-уведомление врачу в течение 30 секунд"

### Методы верификации:
1. **Интеграционное тестирование уведомлений:**
   - Мокировать медицинские устройства
   - Проверить время: получение данных → отправка push

2. **Тестирование граничных значений:**
   - Давление 179/109 → НЕ опасно
   - Давление 180/110 → Опасно
   - Пульс 119 → НЕ опасно
   - Пульс 120 → Опасно

### Медицинская валидация:
- Проконсультироваться с кардиологом по пороговым значениям
- Проверить соответствие клиническим рекомендациям

### Критерии успеха для демо:
- Уведомление приходит за < 30 секунд
- В демо показываем СИНТЕТИЧЕСКИЕ данные (не реальные пациенты)
- На экране врача: красная рамка вокруг "опасного" пациента
